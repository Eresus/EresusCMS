URL-translation engine inactive

Possible reasons:

  1. mod_rewrite module is not exists or not installed
  2. .htaccess file missing or incorrect

Ошибка!

Не работает механизм URL-преобразований

Возможные причины:

  1. На сервере отсутствует или не подключен модуль mod_rewrite
  2. Отсутствует или неверно составлен файл .htaccess
  3. Вы используете сервер несовместимый с Apache
