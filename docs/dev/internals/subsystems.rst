Подсистемы CMS
==============

Ядро
----

Фундамент CMS. Включает в себя такие классы как:

- Eresus_PHP
- Eresus_Kernel
- Eresus_CMS
- Eresus_Accounts
- Sections
- Templates
- WebPage
- WebServer

А также некоторые устаревшие компоненты:

- Eresus
- MySQL
- Eresus Core
