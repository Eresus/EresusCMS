plugin.yml
==========

Пример файла:

.. code-block:: yml

   title: "Название"
   version: "1.00"
   description:
       ru: "Описание"
       en: "Description"
   requires:
       other_plugin:
           min: "1.00"
           max: "1.05"
   settings:
     param1: "value1"
     param2: "value2"


requires
--------

Блок описания необходимых зависимостей.

имя модуля
^^^^^^^^^^

* версии: 1.0 и новее

Указывает зависимость от определённого плагина.

Атрибуты:

* **min** (обязательный) --- минимальная требуемая версия.
* **max** (обязательный) --- максимальная требуемая версия.
* **name** --- имя требуемого плагина.
